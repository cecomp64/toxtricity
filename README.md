# Build
```
make build
```

# Configuration

You should add a `.env` file to the root of this source tree with the following contents:

```
DATABASE_TYPE=sqlite
DISCORD_TOKEN=<your_token>
DISCORD_CLIENT_ID=<your client id>
```

# Run
```
make run
```

# Test
```
npm test
```
